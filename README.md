## Como testar esse projeto?

1. **Primeiro passo**: instalar as dependências

    ~~~sh
     npm install
    ~~~

2. **Segundo passo**: Iniciar servidor local

    ~~~sh
    npm run dev
    ~~~

3. **Terceiro passo**: Acessar aplicação

    [https://localhost:3000](https://localhost:3000)
